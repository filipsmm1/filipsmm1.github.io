# Stylish & Healthy — GitHub Pages Website

A no-image, CSS-only aesthetic website for Stylish & Healthy.

## What is inside

- `index.html` — homepage structure
- `styles.css` — matcha luxury visual design, CSS doodles, animations, responsive layout
- `script.js` — skincare routine builder, affirmation cards, modal popups, slider, counters, reveal animations

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and this `README.md` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. GitHub will give you a live website URL.

## Notes

This version uses no image files. All visuals are CSS-only: journal mockups, skincare bottles, doodle flowers, sparkles, glow gradients, orbit graphics, and animated UI elements.
