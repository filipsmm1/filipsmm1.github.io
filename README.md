# Stylish & Healthy GitHub Pages Website

A static, GitHub Pages-ready website for the Stylish & Healthy brand.

## Files

- `index.html` — homepage structure and copy
- `styles.css` — full luxury visual design
- `script.js` — small skincare routine builder interaction

## How to publish on GitHub Pages

1. Create a new GitHub repository named something like `stylish-and-healthy`.
2. Upload `index.html`, `styles.css`, and `script.js` to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. GitHub will give you a live website link after it deploys.

## Customization

To change product links, edit the `href="#"` values in `index.html`.
To change colors, edit the `:root` variables at the top of `styles.css`.
